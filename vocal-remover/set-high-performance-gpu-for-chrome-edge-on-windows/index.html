Make Windows use your fast graphics chip for your browser

Vocal Remover performs complex math in real time, so it needs a fast graphics chip to work smoothly. Even though this is about audio, modern browsers use the graphics chip behind the scenes to speed up the heavy number‑crunching, and Vocal Remover needs it.

On some computers — especially laptops — Windows may choose the power‑saving graphics chip for your browser. That can cause slowdowns or sound hiccups. This guide tells Windows to use your high‑performance graphics chip for your browser.

Detected browser: —

Set Chrome to use the high‑performance graphics chip

1

Open Graphics settings

Click Open Graphics settings, or go to Settings → System → Display → Graphics.

[placeholder: Windows Settings → Graphics]
2

Add Chrome to the list (only if it’s missing)

In the list, look for Google Chrome (you may need to scroll). If you see it, go to Step 3. If it’s not there, use one of the options below.

A) Try the simple way

  1. Under Add an app, open the drop‑down and choose Microsoft Store app.
  2. Click Browse.
  3. In the list that appears, click Google Chrome. It will be added to the Graphics list.
[placeholder: Dropdown set to Microsoft Store app + Browse]
[placeholder: App list after Browse — click Google Chrome]

B) If you don’t see Chrome there

  1. Under Add an app, open the drop‑down and choose Desktop app.
  2. Click Browse (this opens File Explorer).
  3. At the bottom, in the File name box, paste one of the lines below and press Enter (or click Add). If the first line doesn’t find Chrome, try the next one.
  4. Back in Graphics settings, click Add.
C:\Program Files\Google\Chrome\Application\chrome.exe
C:\Program Files (x86)\Google\Chrome\Application\chrome.exe
%LocalAppData%\Google\Chrome\Application\chrome.exe
[placeholder: Desktop app + Browse]
[placeholder: File name box → paste line → Add → Add in Settings]
3

Choose High performance for Chrome

Back on the Graphics page, find Google Chrome in the list (you may need to scroll). Click it → Options → select High performance → Save.

[placeholder: Choose High performance]
4

Restart the browser — and if needed, your computer

Close Chrome and open it again. If things still feel slow, restart your computer and try once more.

If you don’t see a difference

  • Laptop on battery? Some laptops still prefer the power‑saving graphics chip on battery. Plugging in can help.
  • Update graphics drivers via your graphics chip maker’s app (NVIDIA / AMD / Intel) or Windows Update.
  • Reset: In Graphics settings, set Chrome back to Let Windows decide, then repeat the steps.
